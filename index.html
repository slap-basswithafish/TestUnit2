Windows History

This is a short timeline of the major leaps made in Microsoft's Windows OS overtime.

  • Windows 1.0

    Microsoft. Windows_logo_-_1985-1989. 2013. https://en.wikipedia.org/wiki/Microsoft_Windows_version_history#/media/File:Windows_logo_-_1985-1989.svg

    Being the first Windows "OS" it was very primitive, and was barely an OS. Sold poorly.

  • Windows 95

    Microsoft. 1024px-Windows_Logo_%281992-2001%29. 1992. https://en.wikipedia.org/wiki/Microsoft_Windows_version_history#/media/File:Windows_Logo_(1992-2001).svg

    Windows 95 was released in August 24th 1995, It had some success but didn't get much.

  • Windows Visa

    Microsoft. Windows_Vista_Logo_and_Wordmark. 2023. https://en.wikipedia.org/wiki/Windows_Vista#/media/File:Windows_Vista_Logo_and_Wordmark.svg

    Vista was said to have better security than its predecesors, but wasn't regarded well by the public.

  • Windows 7

    Microsoft. Windows_7_Logo_and_Wordmark. 2023. https://upload.wikimedia.org/wikipedia/en/thumb/2/26/Windows_7_Logo_and_Wordmark.svg/1920px-Windows_7_Logo_and_Wordmark.svg.png

    Windows 7 was a faster and overall better version of Vista. It did well in terms of sales.

  • Windows 10

    Microsoft. Windows_10_Logo. 2014. https://en.wikipedia.org/wiki/Windows_10#/media/File:Windows_10_Logo.svg

    Windows 10 didn't add much except for smoother performance but it still did well.
